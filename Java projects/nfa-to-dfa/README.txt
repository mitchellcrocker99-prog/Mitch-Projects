
## Overview

This Java application takes in an instance of an NFA and converts it
To a DFA

## Compiling and Using

To compile, execute the following command in the p2 directory:
```
$ javac fa/nfa/NFADriver.java
```

Run the compiled class with the command:
```
$ java fa.nfa.NFADriver test/*insert text filename here*
```

Then you will see the output of the resulting DFA from
The file containing the NFA you specified
